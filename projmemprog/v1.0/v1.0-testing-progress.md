## 7. Bugs Fixed During v1.0 Development

| Bug | Root Cause | Fix | Test |
|---|---|---|---|
| Security test false positives | `env_secret` pattern matched "AKIA..." (contains "KEY") | Use cleaner test strings | `test_v12_security_scan_detects_secret` |
| redact_findings trailing newline loss | `splitlines()` + `join()` drops trailing newlines | Detect and preserve trailing newline | `test_redact_findings_*` |
| FrozenInstanceError on SecurityFinding | `frozen=True` prevents mutation | Track `redacted_count` separately | `test_redact_findings_*` |
| mypy "Literal[True] not callable" | `score_files` import shadowed by CLI param | Alias to `_score_files` | mypy strict |
| mypy "Name Config is not defined" | Missing import in `_apply_preset()` | Add `Config` to imports | mypy strict |
| mypy "Name Any is not defined" | MCP server used `dict[str, Any]` | Add `from typing import Any` | mypy strict |
| ruff E501 line too long | `_resolve_effective_mode()` call exceeded 99 chars | Multi-line call | ruff |
| ruff B904 exception chaining | `raise ...` without `from` | Add `from exc` | ruff |
| JSON test parse failure | `result.output` includes stats lines | Use file output (`-o`) | `test_v12_format_json` |
| Init test file not found | `--init` writes to CWD, not `tmp_path` | `monkeypatch.chdir(tmp_path)` | `test_v12_init_generates_config` |
| Config template `_comment` field | Config validation rejects unknown fields | Remove `_comment` from template | `test_v12_init_generates_config` |
